## Smart Inventory Tracker

A premium, intelligent inventory management application that adapts to different environments (Home, Restaurant) to track stock, estimate consumption, and automate purchasing recommendations.

### Features

- **Multi-Context Support**: Track items for Home, Restaurant, Office, etc.
- **Smart Tracking**: Consumption rates and expiry tracking.
- **Automated Shopping List**: Auto-generates list based on low stock.
- **Categories & Search**: Organize items with tags and global search.
- **Data Export**: Export inventory to CSV.

### Getting Started

1.  `npm install`
2.  `npm run dev`
